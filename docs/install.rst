Installation
============

.. rst-class:: lead

   Install **stratix** as a Python package.

----

Stratified Media Electromagnetic Solver


Package install
---------------

.. tab-set::
    :class: outline

    .. tab-item:: uv

        .. code-block:: bash

            uv add stratix

    .. tab-item:: pip

        .. code-block:: bash

            pip install stratix




Development install
-------------------

Clone the repository and install in editable mode:

.. code-block:: bash

    git clone https://github.com/oslumen/stratix
    cd stratix
    uv sync --all-extras
